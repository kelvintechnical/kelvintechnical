## 📁 Featured Research Repositories

### [pytorch-deep-learning-lab](https://github.com/kelvintechnical/pytorch-deep-learning-lab)
**Custom Neural Architectures & Autograd Engineering**

A laboratory for low-level PyTorch experimentation. This repo contains from-scratch implementations of attention mechanisms, custom loss landscapes, and gradient flow analysis tools. Built for researchers who need to debug *below* the API surface.

- Custom autograd functions for non-standard backpropagation
- Architecture search utilities with latency profiling
- Reproducible training pipelines with Weights & Biases integration

---

### [bioinformatics-sequence-toolkit](https://github.com/kelvintechnical/bioinformatics-sequence-toolkit)
**Genomic Data Engineering & Sequence Analysis**

Production-grade utilities for processing large-scale biological datasets. Designed for researchers bridging computational methods and wet-lab validation.

- FASTA/FASTQ parsing with memory-efficient streaming
- Sequence alignment optimization using dynamic programming
- Protein structure prediction preprocessing pipelines
- Integration with NCBI APIs and UniProt databases

---

### [latent-evolution-sim](https://github.com/kelvintechnical/latent-evolution-sim)
**Conceptual Framework for AI-Directed Evolution**

*Dissertation Foundation Work.* A simulation environment modeling how latent space interpolation can guide protein engineering—treating fitness landscapes as optimizable manifolds rather than random search spaces.

- Variational autoencoder architectures for sequence compression
- Fitness landscape modeling in reduced-dimensional spaces
- Directed evolution strategy optimization via reinforcement learning
- Benchmarking against traditional directed evolution methods
